# Styl

To run.

## Instal dependencies.
```bash
npm install
```

## Start mongo.
```bash
sudo mongod
```
## Start development server.
```bash
npm run dev
```
## Navigate to http://localhost:3000

### Stack
- React.js
- Redux
- Node.js
- Express.js
- MongoDB/Mongoose
- Webpack/Babel
