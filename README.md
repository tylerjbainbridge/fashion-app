# Styl

To run.

## Start mongo.
```bash
sudo mongod
```
## Start development server.
```bash
npm run dev
```
## Navigate to http://localhost:3000

### Stack
- React.js
- Redux
- Node.js
- Express.js
- MongoDB/Mongoose
- Webpack for transpiling.
- Babel for ES6 and JSX

- Mocha/Chai
